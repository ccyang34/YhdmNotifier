# 自动化分析与推送集成平台 (Automated Analysis & Notification Hub)

本项目最初作为一个樱花动漫更新自动推送工具起步，现已演变发展为一个**多功能的自动化分析与推送集成平台**。通过集成 GitHub Actions 的定时任务，项目涵盖了 A股 AI 深度分析、ETF 动量轮动策略、期货市场追踪以及自动化的动漫更新推送等多个独立模块。

## 🌟 核心功能模块

### 1. A股市场 AI 综合分析师
利用 AI 大模型（DeepSeek）和公开金融数据（如 akshare/tushare）对 A股市场进行多维度的收盘深度复盘。
- **📈 涨跌停市场分析 (`A股_涨跌停分析.py`)**
  - 自动获取每日涨跌停数据。
  - 通过 AI 分析连板梯队、市场情绪和短线资金接力情况。
  - **触发时间**：工作日 15:30 (北京时间)
- **🏭 行业轮动与资金分析 (`A股_行业轮动分析.py`)**
  - 分析申万行业板块的资金流入流出、涨跌幅分布。
  - 捕捉市场主线和资金切换逻辑。
  - **触发时间**：工作日 15:40 (北京时间)
- **📊 市场宽度快速报告 (`A股_市场宽度报告.py`)**
  - 扫描全市场个股涨跌家数、赚钱效应等宏观市场宽度指标。
  - 提供市场整体健康度评估。
  - **触发时间**：工作日 16:30 & 17:30 (北京时间)

### 2. 量化交易策略追踪
- **🔄 ETF 动量轮动 (`ETF_动量轮动策略.py` / `_v295.py`)**
  - 基于动量策略计算多个核心 ETF 的强度。
  - 自动生成虚拟持仓 (`记录_ETF_虚拟持仓.json`)，计算调仓逻辑。
  - 通过 WxPusher 将调仓建议推送到微信。
  - **触发时间**：工作日 09:25 & 15:30 (北京时间)
- **🛢️ AI 油脂期货分析师 (`期货_油脂分析.py`)**
  - 针对特定商品期货（油脂类）的盘前、盘中、盘后多频次跟踪。
  - 结合宏观和技术面数据输出操作建议。
  - **触发时间**：工作日 08:50, 15:30, 21:00, 23:05 (北京时间)

### 3. 娱乐与资讯推送
- **🎬 动漫更新自动同步 (`动漫_金山文档推送.py`, `动漫_YouTube推送.py`)**
  - 自动抓取动漫更新源，通过 Playwright / BeautifulSoup 等工具解析网页。
  - 将更新数据同步至金山文档，或推送到相关渠道。
  - 记录更新历史以防重复推送 (`记录_动漫_*.json`)。
  - **触发时间**：每天 10:00 - 20:00，每 20 分钟执行一次

## ⚙️ 部署与运行环境

本项目高度依赖 **GitHub Actions** 作为 Serverless 定时调度平台。所有的定时任务都配置在 `.github/workflows/` 目录下。

### 环境变量与 Secrets 配置
如果您想 fork 本项目并自行运行，需要在 GitHub 仓库的 `Settings -> Secrets and variables -> Actions` 中配置以下环境变量：

- `DEEPSEEK_API_KEY`: DeepSeek 大模型的 API 密钥（用于 AI 分析）
- `TUSHARE_TOKEN`: Tushare 金融数据接口的 Token
- `WXPUSHER_APP_TOKEN`: WxPusher 推送服务的应用 Token
- `WXPUSHER_UID`: WxPusher 接收消息的用户 UID
- `APP_TOKEN` / `REPO_TOKEN`: 用于文档同步和 Git 自动提交的凭证

### 依赖环境
各模块依赖的 Python 版本从 `3.9` 到 `3.13` 不等（在 Actions 中自动配置）。
主要使用的第三方库：
- 数据获取：`akshare`, `tushare`, `requests`, `playwright`, `beautifulsoup4`
- 数据处理：`pandas`, `numpy`, `scikit-learn`
- 定时与辅助：`pytz`
